# Contributing

Thank you for your interest in ExpBottleBoost!

## Bug Reports

Bug reports are welcome via [GitHub Issues](https://github.com/sudolifeagain/ExpBottleBoost/issues).

When reporting a bug, please include:
- Minecraft server version (e.g., Paper 1.20.4)
- Plugin version
- Steps to reproduce
- Expected vs actual behavior
- Relevant server logs (if applicable)

## Feature Requests

Feature requests are also welcome via Issues. Please describe the use case and expected behavior.

## Pull Requests

**Pull requests are not accepted at this time.** This is a personal project maintained by a single developer. If you have suggestions, please open an Issue instead.

## Security Vulnerabilities

If you discover a security vulnerability, please report it via Issue or contact the author directly.
